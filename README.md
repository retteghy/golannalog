# GoLANnaLog ‚Äî TUI Network Audit Viewer

Interactive terminal UI for browsing auditd `connect()` syscall logs, built with Go and [Bubbletea](https://github.com/charmbracelet/bubbletea).

Reads `~/.golannalog/audit_connections.csv` (populated by a systemd timer running `export_net_audit.sh`) and displays an interactive, sortable, filterable table enriched with whois data.

## Build

```bash
cd ~/Dev/golannalog
go build -o golannalog
```

## Run

```bash
./golannalog
```

## Install systemd timer

The export script and systemd units are included. To install:

```bash
sudo cp net-audit-export.service net-audit-export.timer /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now net-audit-export.timer
```

## Table columns

| # | Column | Sort key |
|---|--------|----------|
| 1 | **Count** ‚Äî number of `connect()` calls to this IP | `1` |
| 2 | **IP** ‚Äî remote IPv4/IPv6 address | `2` |
| 3 | **Process(es)** ‚Äî unique process names that connected | `3` |
| 4 | **Org** ‚Äî organization from whois (or private IP label) | `4` |
| 5 | **Net** ‚Äî network name from whois | `5` |
| 6 | **Location** ‚Äî city, country from whois | `6` |

## Data source

**CSV format** (`~/.golannalog/audit_connections.csv`):
```
timestamp,process,exe,uid,remote_ip,remote_port
```

Two timestamp formats are supported:
- Human-readable: `DD.MM.YYYY HH:MM:SS.mmm:sequence`
- Unix epoch: `seconds.milliseconds`

The CSV is generated by `export_net_audit.sh` via a systemd timer (`net-audit-export.timer`).

## Whois enrichment

On startup, each unique public IP is resolved using a three-step fallback chain:

1. **`whois`** command ‚Äî extracts OrgName/org-name, NetName/netname, City, Country, descr
2. **`nslookup`** ‚Äî reverse DNS lookup
3. **`ipinfo.io`** ‚Äî HTTP API fallback

Results are cached in `~/.golannalog/whois_cache.json` with a 7-day TTL. Lookups run asynchronously (10 concurrent goroutines) with a progress indicator in the status bar.

### Private IP detection

Private/local IPs (10.x, 172.16-31.x, 192.168.x, 127.x, IPv6 link-local) are labeled automatically:
- **loopback** ‚Äî 127.0.0.1, 127.0.0.53, ::1
- **this machine** ‚Äî matches `hostname -I` output
- **default gateway** ‚Äî matches `ip route show default`
- Reverse DNS via `getent hosts`
- Fallback: "private IP"
