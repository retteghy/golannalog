# GoLANnaLog — TUI Network Audit Viewer

See every outbound connection your Linux machine makes — who it talks to, how often, and who owns that IP — in an interactive terminal UI built with Go and [Bubbletea](https://github.com/charmbracelet/bubbletea).

Reads `~/.golannalog/audit_connections.csv` (populated by a systemd timer running `export_net_audit.sh`) and displays an interactive, sortable, filterable table enriched with whois data.

## Build

```bash
cd ~/Dev/golannalog
go build -o golannalog
```

## Run

```bash
./golannalog
```

## Install systemd timer

The export script and systemd units are included. To install:

```bash
sudo cp net-audit-export.service net-audit-export.timer /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable --now net-audit-export.timer
```

## Key bindings

| Key | Action |
|-----|--------|
| `↑` `↓` | Navigate rows |
| `1`–`6` | Sort by column (press again to toggle asc/desc) |
| `/` | Enter filter mode — type to search across all columns |
| `Enter` | Apply filter and return to table navigation |
| `Esc` | Clear active filter |
| `q` / `Ctrl+C` | Quit |

## Table columns

| # | Column | Sort key |
|---|--------|----------|
| 1 | **Count** — number of `connect()` calls to this IP | `1` |
| 2 | **IP** — remote IPv4/IPv6 address | `2` |
| 3 | **Process(es)** — unique process names that connected | `3` |
| 4 | **Org** — organization from whois (or private IP label) | `4` |
| 5 | **Net** — network name from whois | `5` |
| 6 | **Location** — city, country from whois | `6` |

## Data source

**CSV format** (`~/.golannalog/audit_connections.csv`):
```
timestamp,process,exe,uid,remote_ip,remote_port
```

Two timestamp formats are supported:
- Human-readable: `DD.MM.YYYY HH:MM:SS.mmm:sequence`
- Unix epoch: `seconds.milliseconds`

The CSV is generated by `export_net_audit.sh` via a systemd timer (`net-audit-export.timer`).

## Whois enrichment

On startup, each unique public IP is resolved using a three-step fallback chain:

1. **`whois`** command — extracts OrgName/org-name, NetName/netname, City, Country, descr
2. **`nslookup`** — reverse DNS lookup
3. **`ipinfo.io`** — HTTP API fallback

Results are cached in `~/.golannalog/whois_cache.json` with a 7-day TTL. Lookups run asynchronously (10 concurrent goroutines) with a progress indicator in the status bar.

### Private IP detection

Private/local IPs (10.x, 172.16-31.x, 192.168.x, 127.x, IPv6 link-local) are labeled automatically:
- **loopback** — 127.0.0.1, 127.0.0.53, ::1
- **this machine** — matches `hostname -I` output
- **default gateway** — matches `ip route show default`
- Reverse DNS via `getent hosts`
- Fallback: "private IP"

## Project structure

```
├── main.go                    # Entrypoint: loads CSV, initializes cache, starts TUI
├── data.go                    # CSV parsing, IP aggregation, private IP detection
├── whois.go                   # Whois lookup with JSON cache, async bubbletea integration
├── model.go                   # Bubbletea Model/Update/View, key handling, layout
├── table.go                   # Sorting, filtering, column definitions
├── export_net_audit.sh        # Exports auditd logs to CSV (run by systemd timer)
├── net-audit-export.service   # systemd oneshot service
├── net-audit-export.timer     # systemd timer (every 5 min)
├── go.mod
└── go.sum
```

## Runtime paths

| Path | Purpose |
|------|---------|
| `~/.golannalog/audit_connections.csv` | Connection log (written by export script) |
| `~/.golannalog/whois_cache.json` | Whois lookup cache (written by TUI) |

## Dependencies

- [bubbletea](https://github.com/charmbracelet/bubbletea) — TUI framework
- [bubbles](https://github.com/charmbracelet/bubbles) — table and text input components
- [lipgloss](https://github.com/charmbracelet/lipgloss) — terminal styling

System tools used at runtime: `whois`, `nslookup`, `curl`, `hostname`, `ip`, `getent`.
